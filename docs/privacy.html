FitSynca — Privacy Policy
Last updated: 2025-09-11
Home Usage

FitSynca is an offline-first workout tracker. We do not collect, share, or transmit your personal data to our servers. Your data remains on your device unless you explicitly export or share it using your device’s system share features.

What We Store (On Your Device)

  • Exercises and workouts (sessions, sets, basic metadata) in a local Room database.
  • Optional custom exercise images selected by you (we store a local URI reference, not the image content itself).
  • App settings (theme and basic preferences) via Android DataStore.

Network and Analytics

  • Ads: The app may request network resources to load advertisements (Google Mobile Ads SDK). Ad requests do not include your workout data.
  • No analytics SDKs and no custom telemetry beyond what the ads SDK performs for ad delivery and measurement.

Backups and Import

  • You can export your data to a JSON file using the system file picker. You choose where to store the file (device, cloud provider app, etc.).
  • You can import data from a JSON backup you previously saved.
  • We do not receive or access your backups.

Permissions

  • Photos/Media: used only to let you pick an image for a custom exercise using the system picker. The app stores a URI to the selected image.
  • No location, contacts, or sensor data is collected.

Data Removal

  • In-app delete actions remove the corresponding records.
  • Uninstalling the app removes all app data stored on the device.

Children’s Privacy

FitSynca is intended for general audiences and does not target children. We do not knowingly collect personal data from children.

Changes to This Policy

We may update this policy from time to time. Material changes will be reflected here with an updated date.
