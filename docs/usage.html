FitSynca — Quick Usage Guide

A visual, step‑by‑step overview of the main screens and actions.

This guide covers the basics of using FitSynca to log workouts and manage your data.

Home & Navigation

  • Categories: pick Strength or Cardio exercises from the lists.
  • Bottom banners are ads; your workout data is never sent with ad requests.
  • Use the Back button to return; the app saves only when you tap Save.

Logging a Workout

  • Select an exercise → tap Add Set.
  • Strength: enter reps and weight. Cardio: enter minutes, distance, optional extras.
  • Up to 4 sets per exercise entry. Tap Save Workout to persist.

Custom Exercises

  • In the exercise list, tap Add Exercise to create your own.
  • Optionally pick an image via the system picker; only the image URI is stored locally.

History

  • Open History to review past sessions; tap an item to view or edit.

Export & Import

  • Export: creates a JSON file via the system file picker — you choose where to store it.
  • Import: restores from a JSON backup you previously saved.
  • Tip: keep backups in your preferred cloud drive for safety.

Privacy

  • Your data stays on your device. See the full Privacy Policy.
  • Ads are served as non‑personalized by default in the first release.
